# Android Eros plugin Simple

** Android 插件Simple **

* 插件 简单封装了一个 Module   helloplugin  一个简单的弹出Tost操作。

* 使用

    ```
	var helloplugin = weex.requireModule('helloplugin')
	```

    ```js
	helloplugin.hello()

    ```
